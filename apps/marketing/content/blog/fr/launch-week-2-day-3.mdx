## Présentation de l'API publique de Documenso

Jour. 3. de la Semaine de Lancement 🎉 La mission de Documenso est de créer une plateforme sur laquelle les développeurs du monde entier peuvent construire. Aujourd'hui, nous lançons la première version de notre API publique, et nous sommes très enthousiastes. Étant donné que c'est la première version, nous nous sommes concentrés sur les bases. Avec la nouvelle API, vous pouvez :

- Obtenir des documents (individuels ou tous accessibles)
- Télécharger des documents
- Supprimer des documents
- Créer des documents à partir de modèles
- Déclencher l'envoi de documents pour la signature

Vous pouvez consulter la documentation détaillée de l'API ici :

> DOCUMENTATION DE L'API : [https://app.documenso.com/api/v1/openapi](https://app.documenso.com/api/v1/openapi)

## Tarification

Nous construisons Documenso pour en faire une plateforme ouverte et extensible ; donc l'API est incluse dans tous les plans actuels. L'API est authentifiée via des jetons d'authentification, que chaque utilisateur peut créer sans coût supplémentaire, tout comme les équipes. Les limites existantes s'appliquent toujours (c'est-à-dire le nombre de documents inclus pour le plan gratuit). Bien que nous n'ayons pas encore tous les détails, nous n'avons pas l'intention de facturer l'utilisation de l'API en tant que telle (plutôt les comptes l'utilisant), car nous voulons que vous construisiez sur Documenso sans être étouffé par les coûts de l'API.

> Essayez gratuitement l'API ici : [https://documen.so/api](https://documen.so/api)
