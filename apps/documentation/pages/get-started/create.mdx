# Create your Account

Before starting you need to select the plan this is right for you: https://documen.so/pricing. If you are not sure you can always start with the free plan and upgrade later to receive unlimited signatures in the Individual Plan or add team members via the Teams Plan.

## Creating a new Account

- You can create a new account via the free plan here: https://documen.so/free
- If you want to claim a premium username for the upcoming profiles just register with any name, upgrade to a paid plan and change your username in your profile: https://app.documenso.com/settings/profile
- If you are working with other you can uprade you account to a Team Plan by [creating a team](/get-started/teams)
